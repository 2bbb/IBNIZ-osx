# IBNIZ-osx

simply build tool for [IBNIZ](http://pelulamu.net/ibniz/) on osx

## how to build

```bash
$ ./build.sh
```

* if you didn't install `SDL.framework` then you will be ask sudo password

## binary location

```bash
$ ./bin/ibniz
```

## document location

`./docs/ibniz.txt`
